Note:
-Testing is done live
-Must have a paypal businses account type to setup IPN
-Must have a standard paypal account /w balance to do test transactions.

Paypal IPN Relevant Docs:
https://developer.paypal.com/api/nvp-soap/ipn/
https://developer.paypal.com/api/nvp-soap/ipn/IPNSetup/#id089EG030E5Z

Paypal IPN Link:
https://www.paypal.com/merchantnotification/ipn/preference
https://www.paypal.com/merchantnotification/ipn/history *Can check events here

Steps:
-Install node.js: https://nodejs.org/en
-Install ngrok and register for an account: https://ngrok.com/download
-Follow this yt video from 0:00 to 1:34 for the creation of a localhost server (Can be done using cmd): https://youtu.be/SccSCuHhOw0
-Launch the localhost server using cmd & type "npm run devStart" to where your localhost server file directory is.
-Launch ngrok to host the local server into the internet by typing 'ngrok http 3000'. //Ang port sa ato localhost server is 3000 and ma change sa server.js file.
-Copy the ngrok Forwarding url.
-Open the Web Interface url for monitoring.
-Goto https://www.paypal.com/merchantnotification/ipn/preference.
-Enable IPN by providing the forwarding url generated by ngrok.
-Do live test transactions minimum of 15php thru Paypal.Me/Link or Invoicing.
-Check POST requests made by Paypal into our SERVER in ngrok's Web Interface or in the localhost server console.
